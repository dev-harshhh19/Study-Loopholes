Aim

Part A: Create a simple web page with various server controls to demonstrate the setting and use of their properties.

Part B: Demonstrate the use of List controls (DropDownList, ListBox, CheckBoxList, RadioButtonList).

Background Theory

ASP.NET Server Controls

ASP.NET Web Forms is a framework that allows building dynamic web pages using server controls ‚Äî HTML-like tags prefixed with <asp:> that are processed on the web server before being sent to the browser.

What makes Server Controls special?

  • They run on the server side (marked with runat="server")
  • They automatically maintain ViewState ‚Äî remembering values across postbacks
  • They support event handling in C# code-behind files
  • They can be accessed and manipulated programmatically via their ID

Key Server Controls Used

ControlTagPurpose
TextBox<asp:TextBox>Accepts text input from the user
Button<asp:Button>Triggers a server-side click event
Label<asp:Label>Displays dynamic text output

Server Control vs HTML Control

FeatureServer Control (<asp:TextBox>)HTML Control (<input>)
Processed onServerClient (browser)
Access in C#TextBox1.TextRequest["name"]
ViewStateYes (automatic)No
EventsYes (OnClick, etc.)No server events

How a Postback Works

  1. User fills controls and clicks the Button
  2. Form is posted back to the server (HTTP POST)
  3. The Button_Click event fires in the C# code-behind
  4. Server reads values ‚Üí processes ‚Üí sends updated HTML back

ASP.NET List Controls

List controls allow users to select one or multiple items from a predefined list. They fire server-side events when selection changes and are perfect for forms requiring selection-based input.

List Controls Overview

ControlTagSelectionDescription
DropDownList<asp:DropDownList>SingleItems in a dropdown menu
ListBox<asp:ListBox>Single / MultipleScrollable list of items
CheckBoxList<asp:CheckBoxList>MultipleSet of checkboxes
RadioButtonList<asp:RadioButtonList>Single (exclusive)Set of radio buttons

Key Properties

PropertyDescription
AutoPostBack="true"Posts back to server immediately on selection change
OnSelectedIndexChangedEvent handler called when selection changes
SelectionMode="Multiple"Enables multi-select on ListBox
SelectedItem.TextGets the text of currently selected item
item.SelectedBoolean ‚Äî true if a list item is checked/selected

        <div>
            <h2>Server Control Demo</h2>

            <!-- ASP.NET Server TextBox - accessed via TextBox1.Text in C# -->
            <b>Server TextBox:</b>
            <asp:TextBox ID="TextBox1" runat="server" />

            <!-- ASP.NET Button - OnClick fires Button_Click on server -->
            <asp:Button ID="Button1" runat="server" Text="Click me" OnClick="Button_Click" />

            <br /><br />

            <!-- Plain HTML input - accessed via Request["TextBox2"] in C# -->
            <b>HTML TextBox:</b>
            <input type="text" name="TextBox2" />

            <br /><br />

            <!-- Label to display output -->
            <asp:Label ID="lblOutput" runat="server" ForeColor="Blue"></asp:Label>
        </div>
    </form>
</body>
</html>
servercontrol.aspx.cs
csharp 
using System;
using System.Web.UI;

namespace prac3
{
    public partial class servercontrol : Page
    {
        protected void Button_Click(object sender, EventArgs e)
        {
            // Reading value from ASP.NET server control TextBox directly by ID
            string serverValue = TextBox1.Text;

            // Reading value from plain HTML input using Request object (form collection)
            string htmlValue = Request["TextBox2"];

            // Display both values on the Label control
            lblOutput.Text = "Server TextBox Value: " + serverValue +
                             "<br/>HTML TextBox Value: " + htmlValue;
        }
    }
}

list.aspx.cs
csharp 
using System;
using System.Web.UI;
using System.Web.UI.WebControls;

namespace prac4
{
    public partial class list : Page
    {
        // Fires when user selects a color from DropDownList
        protected void ddlColors_SelectedIndexChanged(object sender, EventArgs e)
        {
            lblSelectedColor.Text = "You selected: " + ddlColors.SelectedItem.Text;
        }

        // Fires when user selects a city from ListBox (multi-select)
        protected void lstCities_SelectedIndexChanged(object sender, EventArgs e)
        {
            string selected = "Selected Cities: ";
            foreach (ListItem item in lstCities.Items)
            {
                if (item.Selected)          // item.Selected is true if checked
                    selected += item.Text + " ";
            }
            lblSelectedCity.Text = selected;
        }

        // Fires when Submit button is clicked (for CheckBoxList)
        protected void btnSubmit_Click(object sender, EventArgs e)
        {
            string selected = "Selected Options: ";
            foreach (ListItem item in chkListOptions.Items)
            {
                if (item.Selected)
                    selected += item.Text + " ";
            }
            lblSelectedOptions.Text = selected;
        }

        // Fires when user selects a gender from RadioButtonList
        protected void rblGender_SelectedIndexChanged(object sender, EventArgs e)
        {
            lblGender.Text = "Selected Gender: " + rblGender.SelectedItem.Text;
        }
    }
}
