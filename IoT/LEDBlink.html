LED Blink - Basic Arduino Programming

Objective: Create a simple circuit that blinks an LED on and off repeatedly using Arduino.

This is the classic "Hello World" program for Arduino. It demonstrates basic digital output control and timing functions.

Required Components:

  • Arduino Uno/Nano
  • LED
  • 220Î© Resistor
  • Breadboard
  • Jumper wires
void setup () {
  pinMode (13, OUTPUT);
}

void loop () {
  digitalWrite (13, HIGH);
  delay(1000);
  digitalWrite (13, LOW);
  delay (1000);
}

How it works:

  • setup(): Runs once when Arduino starts. Sets pin 13 as output.
  • loop(): Runs continuously after setup().
  • digitalWrite(13, HIGH): Turns LED on by sending voltage to pin 13.
  • delay(1000): Waits for 1000 milliseconds (1 second).
  • digitalWrite(13, LOW): Turns LED off.
