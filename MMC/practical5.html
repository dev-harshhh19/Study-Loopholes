Practical 5: Animated GIF Design

Aim

To design an animated GIF by animating text and images using basic animation tools.

Background Information

GIF (Graphics Interchange Format) is a bitmap image format that supports both static and animated images. Animations in GIFs are frame-based, where each frame represents a slight change in the position or state of an object, creating the illusion of motion when played sequentially.

Procedure (Using Adobe Photoshop)

  1. Setting Up Layers

    Import the base image and add text layers. Place each element you want to animate on a separate layer.

  2. Opening the Timeline

    Go to Window → Timeline. Select 'Create Frame Animation' from the dropdown menu.

  3. Creating Animation Frames

    Click the 'New Frame' icon in the Timeline panel. In the new frame, move or transform the objects (e.g., move text from left to right, change opacity).

  4. Setting Duration

    Timing:
    Select all frames and set a delay (e.g., 0.1s or 0.2s).
    Set the looping option to Forever.

  5. Exporting the GIF

    Go to File → Export → Save for Web (Legacy). Choose GIF as the format and ensure 'Animation Looping' is set correctly before saving.
