Practical 10: Color Correction

Aim

To apply color correction to a picture and compare before and after.

Background Information

Color correction is the process of adjusting the colors in an image to make them appear more natural or to achieve a specific creative look. It involves fixing color casts (e.g., removing unwanted yellow or blue tints caused by lighting) and balancing the red, green, and blue channels for a more realistic output.

Procedure (Using Photoshop)

  1. Selecting the Image

    Pick a photograph that has a noticeable color tint or looks 'washed out'. A photo taken indoors under fluorescent light is ideal.

  2. Using White Balance (Auto Color)

    Go to Image → Auto Color (Ctrl+Shift+B). This is a quick way to let Photoshop estimate and fix color casts automatically.

  3. Manual Correction with Color Balance (Ctrl+B)

    Go to Adjustments → Color Balance. Adjust the sliders to move away from unwanted colors (e.g., if the photo is too yellow, move the slider toward Blue).

  4. Using Hue/Saturation (Ctrl+U)

    Hue: Change the overall color tint.
    Saturation: Increase or decrease the intensity of colors.
    Lightness: Adjust how bright or dark the image is.

  5. Before/After Comparison

    Create a duplicate layer (Ctrl+J) before making any changes. Toggle the visibility of the edited layer to compare the results. Use 'Split Screen' by placing both versions side-by-side on a new canvas.

  6. Final Export

    Save as .JPG or .PNG for final presentation.
