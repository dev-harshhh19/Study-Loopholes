Practical 3: Poster Design (2025 Election)

Aim

Design a poster for the 2025 election and show the difference in resolution and quality for Print and Web.

Background Information

Resolution is the amount of detail an image holds. For Web, images are typically 72 PPI (Pixels Per Inch) and use the RGB color model for digital displays. For Print, high resolution (300 DPI - Dots Per Inch) is mandatory for sharpness, and the CMYK color model is used to match physical ink printing.

Procedure

  1. Creating the Layout

    Open an image editing tool like Photoshop. Create a canvas for an A3 poster (29.7 x 42 cm).

  2. Designing the Content

    Add the candidate's photo, election symbol, catchy slogan, and election date (2025). Use layers to keep text and images separate.

  3. Setting Up for Print

    Print Specs:
    Resolution: 300 DPI
    Color Mode: CMYK
    File Format: .PDF or .TIFF

  4. Exporting for Web

    Web Specs:
    Resolution: 72 PPI
    Color Mode: RGB
    File Format: .JPG or .PNG

  5. Comparison

    Observe that the web version is much smaller in file size but looks pixelated when zoomed in, whereas the print version remains sharp but has a significantly larger file size.
