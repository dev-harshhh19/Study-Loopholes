Practical 9: Static IP (3 Routers, 3 PCs)

Aim

Simulate static IP configuration using 3 routers and 3 PCs in Cisco Packet Tracer.

Background Information

This practical demonstrates static IP addressing and routing in a multi-router network. Each router connects to one PC in a separate network. Serial connections are used between routers to enable inter-network communication. Static routes are configured so that all PCs can communicate with each other across different networks.

Procedure

  1. Place Network Devices

    Place 3 PCs and 3 Routers (2911 recommended) on the workspace.

  2. Add Serial Modules

    Turn OFF the routers and add HWIC-2T modules to each to enable serial ports.

  3. Connect the Devices

    PC to Router: Copper Cross-Over cable.
    Router to Router: Serial DCE cable.

  4. Configure IP Addresses

    Assign IPs to PCs and Router interfaces (Gig0/0 and Serial) as specified in the network plan.

  5. Configure Static Routing

    # Router0
ip route 192.168.2.0 255.255.255.0 10.0.0.2
ip route 192.168.3.0 255.255.255.0 10.0.0.2

# Router1
ip route 192.168.1.0 255.255.255.0 10.0.0.1
ip route 192.168.3.0 255.255.255.0 10.0.0.6

# Router2
ip route 192.168.1.0 255.255.255.0 10.0.0.5
ip route 192.168.2.0 255.255.255.0 10.0.0.5

  6. Verify Connectivity

    Ping between PCs in different networks (e.g., PC0 to PC2). Successful replies confirm the routing setup.
