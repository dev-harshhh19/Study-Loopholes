Practical 8: Telnet & FTP

Aim

Configure Telnet and FTP protocols in Cisco Packet Tracer.

Background Information

This practical demonstrates remote access and file transfer services using Telnet and FTP. Telnet allows remote login access to a router, while FTP (File Transfer Protocol) allows file sharing between a client PC and a server. In this setup, the router is configured for Telnet access, and a server device is configured to provide FTP service.

Procedure

  1. Place Network Devices

    Place 2 PCs, 1 Switch (2960), 1 Router (2911 or 1941), and 1 Server on the workspace.

  2. Connect the Devices

    Use Copper Straight-Through cables to connect all devices to the switch.

  3. Configure IP Addresses

    Router (Gig0/0): 192.168.20.1
    PC0: 192.168.20.2
    PC1: 192.168.20.3
    Server: 192.168.20.10

  4. Configure Telnet on Router

    enable
configure terminal
username admin password admin123
line vty 0 4
login local
transport input telnet
exit

  5. Configure FTP on Server

    1. Click Server ‚Üí Services ‚Üí FTP.

    2. Turn FTP service ON.

    3. Add user: Username ftpuser, Password ftp123.

  6. Verify Connections

    Telnet: From PC0 Command Prompt: telnet 192.168.20.1.

    FTP: From PC1 Command Prompt: ftp 192.168.20.10.
