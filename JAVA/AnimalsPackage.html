Animal.java - Interface Definition

Interface defining contract for Animal behaviors

package Animals;

public interface Animal {
    void sound();
    void eat();
}

Dog.java - Interface Implementation

Dog class implementing Animal interface

package Animals;

public class Dog implements Animal {
    public void sound() {
        System.out.println("Dog barks");
    }

    public void eat() {
        System.out.println("Dog eats bones");
    }
}

Main.java - Interface Usage

Main class demonstrating interface usage

import Animals.Dog;

public class Main {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.sound();
        d.eat();
    }
}
